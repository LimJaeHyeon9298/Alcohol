# Alcohol
beer description with using punk api


# :sparkles: 프로젝트 소개

-  외국의 맥주와 맥주 마실때의 팁, 페어링하기 좋은 안주를 알려주는 앱
 
 - MVC Architecture 사용



## :people_holding_hands: 멤버구성

- 개인 프로젝트 진행

## :gear: 개발 환경 

- xcode 14.3.1

- framework : Snapkit 5.6.0 , KingFisher 7.9.1 

- API : Punk API

  https://punkapi.com/

TableView를 내리면서 각 나라의 맥주의 특징, 설명 등을 알 수 있다.
